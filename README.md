# Grocery List Manager

this is a simple grocery list manager that allows someone to create a grocery list and add or remove items to the list also allows them to view it aswell

## How to Run

1. Make sure Python is installed on your computer
2. Open a terminal or command prompt inside of the project folder
3. Run the Python file with: Python grocery_list.py

## Features

* Allows a grocery list to be created
* Allows stuff to be added to the grocery list and stored
* Allows stuff to be removed from the grocery list aswell
* Has a go back system for user can go back and after adding something or removing goes back to the main menu to decide to add or remove again. or if decided just to end aswell
* Also allows viewing of the list if decided aswell
