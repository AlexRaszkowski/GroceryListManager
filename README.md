# Grocery List Manager

this is a simple grocery list manager that allows someone to create a grocery list,
add or remove items to the list or view it.

## How to Run

1. Make sure Python is installed on your computer
2. Open a terminal or command prompt inside the project folder
3. Run the program:

```bash
python grocery_list.py
```

## Features

* Allows a grocery list to be created
* Add stuff to list
* Remove stuff to list
* Go back system
* View List
